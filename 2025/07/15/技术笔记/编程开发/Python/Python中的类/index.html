四、三种方法类型

Python 类中有三种方法：实例方法、类方法、静态方法。

4.1 三种方法对比

类型 装饰器 第一个参数 访问能力
实例方法 无 self（实例） 实例属性、类属性
类方法 @classmethod cls（类） 只能访问类属性
静态方法 @staticmethod 无 都不能访问

4.2 实例方法（最常用）

4.3 类方法

需要 @classmethod 装饰器，第一个参数是 cls（类本身）。

4.4 静态方法

需要 @staticmethod 装饰器，没有 self 也没有 cls。

4.5 self vs cls

4.6 什么时候用哪个？

4.7 选择总结

方法类型 何时使用
实例方法 需要访问或修改具体对象的属性
类方法 需要访问类属性，或作为工厂方法创建对象
静态方法 逻辑上属于这个类，但不需要访问类或实例

简单记忆：

  • 操作 self.xxx → 实例方法
  • 操作 cls.xxx 或工厂方法 → 类方法 @classmethod
  • 只是个工具函数 → 静态方法 @staticmethod
